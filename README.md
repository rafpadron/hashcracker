# 🔐 Hash Cracker

Ferramenta em Python para quebra de senhas a partir de hashes, desenvolvida
para fins educacionais no estudo de segurança da informação.

## Funcionalidades
- Identifica e quebra hashes (MD5, SHA-1, SHA-256, etc.)
- Suporte a ataques por dicionário (wordlist)

## Tecnologias
- Python
- Bibliotecas de hashing (hashlib)

## Como executar
```bash
python hashcracker.py
```

## Objetivo
Projeto criado para entender na prática como funcionam os algoritmos de hash
e a importância de senhas fortes — base para estudos em Segurança Cibernética.

⚠️ Uso exclusivamente educacional.
